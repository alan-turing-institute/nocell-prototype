2.2.1 Running a nocell program

Nocell programs can be used to generate spreadsheets via a selection of helper scripts. For example, in your shell, running

  nocell-to-ods foo.nocell

Assumes the "scripts" directory is in your path: see Installation

will produce "foo.ods", an OpenDocument spreadsheet, assuming that "foo.nocell" exists and is a valid nocell program.

A nocell program is really a Racket module, and may be required by another Racket program. When this is done, anything provided by your nocell module is available as a cell data structure. Some advice on this mode of usage is provided in the next section.
